# ランダムフォレスト

## ランダムフォレストとは

ランダムフォレストはアンサンブル学習の一種で、バギングというアンサンブル学習の方法と複数の決定木を組み合わせたモデルです。
前回の決定木では木が一つであり、木の深さが深くなるほど過学習になりやすいという問題がありました。
ランダムフォレストでは、複数の決定木を組み合わせることで、過学習を防ぎます。

:::tip アンサンブル学習とは
複数の機械学習モデルを組み合わせることで、個々のモデルの誤差を相殺し、より優れたモデルを構築する手法です。
アンサンブルには、バギング、ブースティング、スタッキングなどがあります。
:::
