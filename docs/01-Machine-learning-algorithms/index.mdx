# 機械学習アルゴリズム

## 機械学習の種類
一般的な機械学習アルゴリズムは以下のように分類されます。
- 教師あり学習
- 教師なし学習
- 強化学習

上から順にその概要と応用例を見てみましょう

## 1. 教師あり学習

入力データとそれに対応する出力データ（ラベル）のペアを用いてモデルを学習します。学習済みモデルを用いて、新しい入力データに対して出力データを予測します。その応用例を見てみましょう。

### 分類(Classification)

#### 代表的なアルゴリズム
- ロジスティック回帰
- サポートベクターマシーン(SVM)
- 決定木
- ランダムフォレスト

### 回帰(Regression)
- 線形回帰
- リッジ回帰
- Lasso回帰
- 決定木
- ランダムフォレスト

#### 代表的なアルゴリズム

## 2. 教師なし学習
教師なし学習では、ラベルなしのデータを用いてモデルを学習します

### クラスタリング

#### 代表的なアルゴリズム
- k-means
- 階層的クラスタリング
- DBSCAN

### 次元削減
#### 代表的なアルゴリズム
- 主成分分析 (PCA)
- t-SNE
- UMAP

### 異常検知

#### 代表的なアルゴリズム
- One-class SVM
- Isolation Forest
- Autoencoders

## 3. 強化学習
学習では、エージェントが環境と相互作用し、報酬を最大化する行動を学習します。主にゲームやロボット制御など、決定過程が重要な問題に適用されます。

#### 代表的なアルゴリズム
- Q学習
- SARSA
- Deep Q Network
- Policy Gradient
- Actor-Critic
