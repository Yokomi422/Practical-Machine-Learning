# 機械学習の一連の流れと用語集

## 初めに

この章では、伝統的な機械学習の一連の流れを紹介します。
そして最後に機械学習に関わる用語を示しましたので、適宜参照ください。

## 機械学習の一連の流れ

### ① データの収集

最初のステップは、モデルの訓練と評価に使用するデータを収集することです。データは様々な形で存在し、それは観測された値（温度、重量など）、カテゴリ（色、ブランドなど）、またはテキスト（レビュー、記事など）など、あらゆる形を取る可能性があります。データはさまざまな情報源から取得できます。それは公開されているデータセット、企業の内部データベース、あるいはウェブスクレイピングといった手法でインターネットから収集した情報など、さまざまです。


### ② データの分析方法を決める

データ収集後、次に適切なデータ分析方法を決定します。これは、データの種類（数値、カテゴリ、テキストなど）、目的（分類、回帰、クラスタリングなど）、および特定の問題の複雑さによります。これには、探索的データ分析（EDA）が含まれ、これによりデータの性質、パターン、外れ値を理解することができます。

### ③ モデルとその性能評価指標を決める

問題とデータの性質に基づいて、適切な機械学習モデルを選択します。これには、線形回帰、ロジスティック回帰、決定木、ランダムフォレスト、サポートベクトルマシン、ニューラルネットワークなどがあります。モデルの選択は、問題の複雑さ、データの量、必要な精度、トレーニングと予測の時間など、多くの要素によります。
モデルの性能を評価するために、適切な評価指標を選択する必要があります。これは、モデルが解決しようとしている特定の問題によります。分類問題では、精度、適合率、再現率、F1スコアなどの指標が一般的に使用されます。一方、回帰問題では、平均絶対誤差（MAE）、平均二乗誤差（MSE）、またはR^2スコアなどが使用されます。
